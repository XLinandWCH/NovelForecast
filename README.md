# 马良 - 小说创作助手 (MaLiang - Novel Creation Assistant)

欢迎使用“马良 - 小说创作助手”！本应用旨在通过 AI 和 RAG（检索增强生成）技术，帮助您更好地分析、理解和创作小说。它采用了左右分栏的布局，左侧为沉浸式的工作区（支持文本、思维导图、关系图谱视图），右侧为 AI 对话控制台。

## 🌟 核心功能

1. **多模型支持 (AI & RAG)**
   - 支持接入多种大模型 API，包括但不限于：Gemini, OpenAI, DeepSeek, GLM, Qwen, 魔塔社区 (ModelScope)。
   - 完美支持本地大模型部署方案：**LM Studio**, **Ollama**, **Nexa AI**。
   - 可以在设置中独立配置对话模型和 RAG（检索增强生成）模型。

2. **知识库管理与 RAG 解析**
   - 支持上传 `.txt`, `.docx`, `.pdf` 格式的文件。
   - 自动解析文件内容并提取为 RAG 上下文，让 AI 基于您的设定进行预测。
   - 实时显示 RAG 解析状态（未解析、解析中、解析完成、解析失败）。

3. **多视图沉浸式分析**
   - **文本视图**：直接阅读上传的原始文本。
   - **导图视图**：将文本内容自动转换为清晰、现代化的思维导图（支持 Markdown 标题解析，支持缩放拖拽）。
   - **关系图谱**：AI 自动提取文本中的人物关系，生成直观、可交互的动态关系网络（区分主角、反派、配角，支持高亮交互）。

4. **流式预测与工作模式**
   - **工作模式**：向 AI 发送预测指令后，结果不会挤在聊天气泡中，而是会自动在左侧工作区新建一个文件，并以**流式（打字机效果）**实时输出。

## 🚀 快速开始指南

### 第一步：配置 AI 和 RAG 模型 (非常重要)

为了让应用正常工作，您需要配置大语言模型 (AI) 和检索模型 (RAG)。点击左下角的 **“设置” (齿轮图标)** 进入配置页面。

#### 1. AI 模型配置 (用于对话和生成内容)
- **提供商**：选择您想使用的 AI 服务商。
- **模型名称**：填写对应的模型名称（例如：`gpt-4o`, `deepseek-chat`）。
- **API Base URL**：填写服务商的 API 地址。系统会根据您选择的提供商自动填入默认地址。
- **API Key**：填写您在服务商处申请的 API 密钥。**注意：如果您使用的是本地模型（如 LM Studio 或 Ollama），此项可以留空。**

#### 2. RAG 检索配置 (用于文档解析和向量化)
- **提供商**：选择用于生成文本向量的模型服务商。
- **模型名称**：填写对应的 Embedding 模型名称（例如 OpenAI 的 `text-embedding-3-small`，或者本地模型的名称）。
- **API Base URL**：填写服务商的 API 地址。
- **API Key**：填写 API 密钥（本地模型可留空）。

---

### 第二步：解决本地 RAG 模型的连接问题 (针对 GitHub 下载源码或本地部署的用户)

如果您在设置中配置了**本地的 RAG 模型**（例如运行在您自己电脑上的 LM Studio 或 Ollama，URL 通常是 `http://localhost:1234/v1` 或 `http://localhost:11434/v1`），并且您是通过**线上链接 (HTTPS)** 访问本应用的，您可能会遇到 **RAG 解析失败** 的问题。

这是因为浏览器的安全机制（Mixed Content）禁止 HTTPS 网页直接请求 HTTP 的本地服务。

**解决方法：使用内网穿透工具 (推荐 ngrok)**

您需要将本地的 HTTP 服务暴露为一个 HTTPS 链接。操作非常简单，只需 1 分钟：

**Windows 用户：**
1. 下载 ngrok：[点击这里下载 Windows 版](https://bin.equinox.io/c/bNyj1mQVY4c/ngrok-v3-stable-windows-amd64.zip)，解压后得到一个 `ngrok.exe` 文件。
2. 双击打开 `ngrok.exe`，在弹出的黑框框里输入以下命令并回车：
   - 如果您用的是 Ollama：`ngrok http 11434`
   - 如果您用的是 LM Studio：`ngrok http 1234`
3. 屏幕上会出现一行类似 `Forwarding https://xxxxxx.ngrok-free.app -> http://localhost:11434` 的信息。
4. 复制这个 `https://xxxxxx.ngrok-free.app` 链接。
5. 回到本应用的“设置”页面，在 **RAG Base URL** 中填入：`https://xxxxxx.ngrok-free.app/v1` （注意加上 `/v1`）。

**Mac 用户：**
1. 打开终端 (Terminal)，输入命令安装 ngrok：`brew install ngrok/ngrok/ngrok`
2. 安装完成后输入命令启动：`ngrok http 11434` (端口号根据您的本地软件而定)
3. 复制生成的 `https://...` 链接，并在本应用的设置中填入（记得加上 `/v1`）。

配置完成后，您上传文件时，左侧文件列表下方就会显示绿色的“RAG 解析完成”状态了！

---

### 第三步：上传文件与解析

1. 点击左侧边栏的 **“上传文件”** 按钮。
2. 选择您的 txt 格式小说文件。
3. 文件上传后，系统会自动开始 RAG 解析（分词和向量化）。您可以在左侧文件列表中看到进度条。
4. **状态说明**：
   - `未进行 RAG 解析` (黄色)：表示您尚未在设置中配置 RAG URL。
   - `正在进行 RAG 解析...` (带进度条)：系统正在处理文件。
   - `RAG 解析完成` (绿色)：文件已成功向量化，可以用于精准问答。
   - `RAG 解析失败` (红色)：解析过程中出错，请检查您的网络或 RAG 设置（参考上文的 ngrok 教程）。

---

### 第四步：开始创作与分析

- **对话问答**：在右侧聊天窗口中，直接向 AI 提问。AI 会自动检索您上传的知识库文件，并给出基于原文的准确回答。
- **生成关系图谱**：
  1. 在左侧选中一个已上传的文件。
  2. 点击中间区域顶部的 **“关系图谱”** 按钮。
  3. 点击 **“基于当前文本生成图谱”**。AI 将分析文本并绘制出精美的人物关系网。
- **查看思维导图**：
  1. 选中文件。
  2. 点击顶部的 **“导图视图”**。系统会自动将文本结构化为思维导图。
- **工作模式沉淀内容**：
  1. 在右侧聊天输入框下方，将模式切换为 **“工作模式”**。
  2. 发送您的指令。
  3. AI 的回复将**自动在左侧知识库中生成一个新的 txt 文件**，方便您后续编辑和管理。

## ⚙️ 如何配置自定义 AI 图标

侧边栏左上角的 AI 图标默认显示文字 "AI"。如果您想替换为自己的图片：

1. 准备一张正方形的图片（推荐 PNG 或 SVG 格式）。
2. 将图片重命名为 `ai-logo.png`。
3. 将该图片放入项目的 `public` 文件夹中。
4. 刷新页面，系统会自动加载并显示该图片。如果图片加载失败，会自动回退显示 "AI" 文字。
